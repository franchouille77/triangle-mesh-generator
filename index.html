About Triangle Mesh Generator

A lightweight, browser-based tool for generating equilateral triangle mesh patterns inside a rectangle. Designed for precision work in laser cutting, CNC routing, architectural panels, and graphic design.

  • Real-time preview with SVG rendering
  • Configurable edge length & stroke width
  • Export to SVG and DXF
  • Screen calibration for true 1:1 scale
  • Pan, zoom & fit navigation

No install required — runs entirely in your browser. Zero dependencies.
